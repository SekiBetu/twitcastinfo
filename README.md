# twitcastinfo
获取twitcasting频道的历史直播信息（是否有录播，直播链接，直播标题，直播时间）

### 方法  
`./twitcastinfo.sh "频道号码"`  
### 示例  
`./twitcastinfo.sh "c:annuuuu_cas" > twitcastinfo_c:annuuuu_cas.txt`
### 输出格式示例
是否有录播，直播链接，直播时间，直播标题
